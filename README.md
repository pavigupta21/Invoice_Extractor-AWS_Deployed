# Invoice Extractor (AWS Deployed)

A web application that extracts key information from PDF invoices using
Amazon Textract and a serverless AWS backend.

## Tech Stack
- Frontend: React (Vite)
- Backend: AWS Lambda, API Gateway
- OCR & Extraction: Amazon Textract
- Database: DynamoDB

## Notes
- Deployed for demonstration purposes
- Invoice extraction accuracy depends on document format
